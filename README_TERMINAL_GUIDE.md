# How to Open Terminal (Terminal Kaise Kholein)

There are 3 main ways to open the terminal in Windows/VS Code:

## Method 1: In Visual Studio Code (Yahan se sabse easy hai)
1.  **Top Menu Bar:** Click on `Terminal` -> `New Terminal`.
2.  **Shortcut Key:** Press `` Ctrl + ` `` (backtick key, `Tab` ke upar wala button).
3.  **Bottom Panel:** Neeche ek naya panel khulega jisme aap commands type kar sakte hain.

## Method 2: Command Prompt (CMD) via File Explorer
1.  Go to your project folder: `c:\Users\ishwa\aiml enable medicine management system`
2.  Click on the **Address Bar** (Upar jahan path likha hota hai).
3.  Type `cmd` inside the address bar and press Enter.
4.  A black window will open directly in this folder.

## Method 3: PowerShell (Start Menu)
1.  Open Start Menu.
2.  Type "PowerShell".
3.  Open it and type:
    `cd "c:\Users\ishwa\aiml enable medicine management system"`
4.  Press Enter to go to your project folder.

---

**Tip:** **Method 1 (VS Code Terminal)** sabse best hai kyunki aap code edit karte hue side-by-side terminal dekh sakte hain.

Once terminal is open, verify you are in the correct folder by typing:
`cd` (Windows) or `pwd` (Linux/Mac)
It should show: `c:\Users\ishwa\aiml enable medicine management system`
